### 缩进与空格

  - 2 空格缩进。

  - 选择器后空格一位，属性的冒号后空一位。

  - 无论选择器有多长，在外联样式表中时，选择器中的内容都应该每项独占一行。

  - 多个选择器叠加时，可以在同一行，中间增加一个空格。如：`.session, .footer { ... }`

<br/>

### 风格

  - 在 `CSS Module` 中，命名应尽可能简短，使用词而非缩写。

  - 避免使用 ID 选择器。

  - 避免使用 `important`。

  - 禁止在子组件或子模块中修改父级样式或全局样式。

  - 16 进制颜色值使用小写。

  - 在使用各类 CSS 预处理工具时，请勿多层嵌套 (>3+)，这会造成权重的累加，也不利于阅读。
